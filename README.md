# かんじのもり 🦉

小学1〜3年生の配当漢字**440字**をぜんぶ収録した、親子で遊べる漢字ゲームです。
はちみつかるた・ひらがじゃんにヒントをもらったオリジナルルールで、6歳ごろから遊べます。

## あそびかた

| モード | 内容 |
| --- | --- |
| 🎴 **かるた** | 「がっこう 🏫」のよみ札を見て、`学□` の □ に入る漢字カードをタップ（10問） |
| ⚔️ **かるた たいせん** | ふたり用。同じ札を左右に表示し、早く正しくタップした方が札を取る。先に5枚で勝ち！ |
| 🧩 **ことばづくり** | 「はなび 🎆」を見て「花」→「火」の順に牌を選んで、ことばを完成（8問） |
| 🀄 **かんじパズル** | 2〜4人対戦。手札の漢字を7×7の盤に置き、縦横で二字熟語を作って文字数分の得点。作れないときは1〜3枚交換してパス。1人8ターンで最高得点が勝ち。**じしょモード**（約620語の辞書でアプリが判定・置けるマスが光る）と**フリーモード**（どこでも置けて、熟語になったかは⭕❌でみんなで判定）を選べる |
| 📖 **かんじずかん** | 正解して集めた漢字に⭐がたまるコレクション。タップすると読み上げ |

ホーム画面で「１ねんせい / ２ねんせい / ３ねんせい / ぜんぶ」を切りかえられます。
集めた⭐はブラウザ（端末）に保存されます。

## iPhone / iPad でオフラインで遊ぶ（飛行機対応✈️）

1. このフォルダを静的ホスティングに公開します（いちばん簡単なのは GitHub Pages）:
   - GitHub にリポジトリを push → Settings → Pages → Branch: `master` / `(root)` → Save
   - 数分で `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます
2. iPhone / iPad の **Safari** でそのURLを開く
3. 共有ボタン → **「ホーム画面に追加」**
4. 一度ホーム画面のアイコンから起動すれば、Service Worker が全ファイルをキャッシュするので、**以後は機内モードでも遊べます**

> 搭乗前に一度、Wi-Fi がある状態でアプリを起動しておくのが確実です。

## 開発

ビルド不要のバニラ HTML/CSS/JS です。ローカル確認は任意の静的サーバーで:

```
python -m http.server 8000   # なんでもOK
```

`file://` で index.html を直接開いても遊べます（Service Worker だけ無効）。

### ファイル構成

- `index.html` — 全画面（ホーム / かるた / たいせん / ことばづくり / ずかん / けっか）
- `js/data.js` — 漢字データ440字 + 二字熟語リスト
- `js/words.js` — かんじパズル用の二字熟語辞書（約620語）
- `js/app.js` — かるた・たいせん・ことばづくり・ずかんのロジック
- `js/puzzle.js` — かんじパズル（対戦ボードゲーム）のロジック
- `css/style.css` — スタイル
- `sw.js` / `manifest.webmanifest` / `icons/` — PWA（オフライン対応）
